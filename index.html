Tudo sobre HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que signigica Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas Web. Documentos HTML podem ser interpretados por navegadores. - wikipedia.org - acesso 09/2016

Tecnologia - Realidade Aumentada

De forma simples Realidade Aumentada é uma tecnologia que permite que o mundo virtual seja misturado com o real, possibilitando maior interação e abrindo uma nova dimensão na maneira como nós executamos tarefas, ou mesmo as que incubimos às máquinas. Assim se você pensava que objetos pulando para fora da tela eram elementos de filmes de ficção científica, está na hora de mudar seus conceitos.
